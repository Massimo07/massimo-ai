# /utils – Massimo AI

Questa cartella raccoglie tutte le utility e i toolbox fondamentali per Massimo AI:

- Generazione ID, token, hash sicuri
- Utility di tempo/data/timestamp
- Validazione e invio email, manipolazione stringhe
- Funzioni di sicurezza e cifratura password
- Gestione file e path, logging avanzato
- Funzioni di testing e mock data
- Pronto per estensioni plug-in, custom helper, AI utilities

**Questo è il vero coltellino svizzero dell’intero sistema.**
