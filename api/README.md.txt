# /api – Massimo AI

Espone tutte le API RESTful (e gRPC se richiesto) del sistema, documentate e versionate.

- Sicurezza JWT/OAuth2, CORS, throttling, logging
- Versioning (v1, v2…)
- Documentazione automatica OpenAPI/Swagger
- Routing per tutte le entità principali: utenti, mondi, agenti, abbonamenti, pagamenti, automazioni, analytics
- Pronto per estensione gRPC, WebSocket e GraphQL
- Test automatici inclusi
